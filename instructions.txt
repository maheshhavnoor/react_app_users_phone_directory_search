This is a react application where REST API is consumed to fetch the records and display it in table.

For testing purpose the data is loaded from local. However Code is written to fetch the records from MySQL database.

for running this application, you need download it and install the dependencies in your local system.
Below are the dependencies to be installed to run this app.
    "@testing-library/jest-dom"
    "@testing-library/react"
    "@testing-library/user-event"
    "axios": "^0.21.1",
    "bootstrap": "^4.5.3",
    "date-fns": "^2.16.1",
    "react": "^17.0.1",
    "react-dom": "^17.0.1",
    "react-scripts": "4.0.1",
    "react-table": "^7.6.3",
    "react-table-sticky": "^1.1.3",
    "styled-components": "^5.2.1",
    "web-vitals": "^0.2.4"
 To install above dependencies, just runt the command npm install {dependency name}   
 
 
 once all done, we can run the application by the command
 npm start
